Что мы используем:

1. Датчик газа(MQ135)
2. Датчик температуры и давления(BME280)
3. Arduino UNO
4. Ethernet shield (ENC28J60)

Схема подключения:
1. Ethernet Shield:

    ENC28J60 |  Arduino UNO
       VCC          5V
       GND	        GND
       SCK          D13
       SO	          D12
       SI	          D11
       CS           D10


2. MQ135:
      MQ135 |  Arduino UNO
       VCC          5V
       GND	        GND
       A0           A0

3. BME280
      MQ135 |  Arduino UNO
       VCC          3.3V
       GND	        GND
       SCL          A5
       SDA          A4


Данные отправляются в формате: 051210074B4657414F43544A45444B42515F565E5F47555E515F544A41 - Это hex-код, который был ранее зашифрованный xor шифрованием. 
Исходные данные выглядят так: data:132.00,747.27,27.25

Где 132 - уровень газа, 747 - давление, 27 - температура.


При запуске Arduino UNO, он отправляет запрос на сервер "GET_CRITICAL" -> Получить критический значения, сервер должен ответить на запрос и отправить крит.значения.

После идет снятие с датчиков показания, если они превышают критические, то включается режим "ALERT" -> данные отправляются каждую секунду.
А если нет, то данные отправляются в спокойном режиме каждые 30 секунд.
